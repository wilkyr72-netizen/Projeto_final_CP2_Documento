01 — Visão Geral

O que o projeto faz

O Alerta do Fogo monitora regiões geográficas específicas em busca de focos de incêndio. Ao detectar um evento, o sistema classifica o risco automaticamente, registra os dados em planilha e dispara uma mensagem formatada num grupo do Discord — com coordenadas, nível de perigo e recomendações.

Sistema Ativo Monitoramento Contínuo
🗺️

Mapeamento de Focos

Detecta e localiza incêndios em pontos georreferenciados, retornando latitude, longitude e região afetada.

🤖

Classificação Inteligente

Algoritmo KNN analisa os dados do foco e classifica automaticamente o nível de risco: Perigo ou Seguro.

💬

Alertas via Discord

Mensagens formatadas com template pré-definido são enviadas automaticamente ao grupo de monitoramento.

📊

Registro em Planilha

Todos os eventos são armazenados em planilha com timestamp, coordenadas e classificação para análise posterior.

03 — Fluxo

Como Funciona

01

Detecção de Evento

O n8n monitora a fonte de dados de focos de calor (sensor, API pública ou feed) e aciona o workflow ao identificar um novo evento na área configurada.

02

Extração de Coordenadas

Latitude, longitude, área estimada e intensidade do foco são extraídos e normalizados para o processamento seguinte.

03

Classificação por KNN

O código JavaScript executa o algoritmo KNN comparando o novo foco com eventos históricos. O resultado é uma classificação binária: PERIGO ou NÃO PERIGO.

04

Registro na Planilha

O evento é salvo com todos os dados: data, hora, coordenadas, classificação e fonte da detecção. Serve como base de treino para futuras classificações.

05

Alerta no Discord

O n8n seleciona o template correto (perigo ou seguro) e envia a mensagem ao grupo Discord com todas as informações do evento de forma clara e padronizada.

06 — Algoritmo

Classificação por KNN

O K-Nearest Neighbors compara o novo foco de incêndio com eventos históricos na planilha. Com base nos K vizinhos mais próximos (por distância geográfica e intensidade), o sistema vota na classificação majoritária.

// Lógica simplificada de classificação — JavaScript (n8n Function Node)

// Dados do evento atual (vindos do nó anterior)
const novoFoco = {
  lat:       -3.119,
  lon:       -60.021,
  intensidade: 87.4,   // FRP — Fire Radiative Power
  area_ha:   120
};

// Histórico da planilha (lido pelo nó Google Sheets)
const historico = $input.item.json.historico;
const K = 5;

// Distância euclidiana normalizada
function distancia(a, b) {
  return Math.sqrt(
    (a.lat - b.lat)**2 +
    (a.lon - b.lon)**2 +
    ((a.intensidade - b.intensidade) / 100)**2
  );
}

// Ordenar por proximidade e pegar os K mais próximos
const vizinhos = historico
  .map(h => ({ ...h, dist: distancia(novoFoco, h) }))
  .sort((a, b) => a.dist - b.dist)
  .slice(0, K);

// Votação majoritária
const votos = vizinhos.filter(v => v.classificacao === 'PERIGO').length;
const resultado = votos > K / 2 ? 'PERIGO' : 'NAO_PERIGO';
const confianca = (Math.max(votos, K - votos) / K * 100).toFixed(0);

return [{ json: { ...novoFoco, resultado, confianca } }];
